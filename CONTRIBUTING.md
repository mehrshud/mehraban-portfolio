# Contributing to Mehraban Portfolio
=====================================

Thank you for considering contributing to the Mehraban Portfolio project. This document outlines the guidelines and best practices for contributing to the project.

## Forking the Repository
-------------------------

To contribute to the Mehraban Portfolio project, you will need to fork the repository. Here's a step-by-step guide:

1. Go to the [mehraban-portfolio](https://github.com/mehraban/mehraban-portfolio) repository on GitHub.
2. Click the "Fork" button in the top-right corner of the page.
3. Select your GitHub account as the location to fork the repository.
4. Wait for the fork to complete.

## Branch Naming
----------------

When creating a new branch for your contribution, please use the following naming convention:

* `feature/<feature-name>` for new features
* `fix/<bugfix-name>` for bug fixes
* `docs/<doc-update>` for documentation updates
* `chore/<chore-update>` for miscellaneous updates

## Commit Style
----------------

Please follow the standard GitHub commit message guidelines:

* Use the imperative mood (e.g., "Add feature" instead of "Added feature")
* Keep the first line concise (<50 characters)
* Use a blank line to separate the summary from the body
* Use bullet points for multiple changes

Example:
```markdown
Add new feature

* Implement new functionality
* Update documentation
```

## Pull Request Template
-------------------------

When submitting a pull request, please use the following template:

### PR Template
```markdown
## Description
Please describe the changes made in this PR.

## Changes
* List the specific changes made in this PR.

## Related Issues
* List any related issues or PRs.

## Testing
* Describe how the changes were tested.

## Screenshots (if applicable)
* Add screenshots of the changes (if applicable).
```

## Code of Conduct
------------------

The Mehraban Portfolio project adheres to the standard GitHub [Code of Conduct](https://github.com/mehraban/mehraban-portfolio/blob/main/CODE_OF_CONDUCT.md). By contributing to the project, you agree to abide by its terms.

Thank you for your contribution to the Mehraban Portfolio project!
